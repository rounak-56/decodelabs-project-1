# **DecodeLabs Summer Internship - Project 1**

A modern, static landing page designed for a fictional travel service "Travel India", this project serves as the foundational milestone for the DecodeLabs frontend internship, focusing heavily on semantic HTML structure and complex CSS grid layouts to showcase incredible destinations across the country.

> **[View Live Demo Here](https://rounak-56.github.io/Decodelabs-Project-1/)**
<br>

## Project Objective
The goal of this phase was to construct a rigid, high-fidelity desktop web layout from scratch without relying on external CSS frameworks. The primary focus was placed on visual hierarchy, typography, and precise element positioning. 

<br>

## Tech Stack & Skills Highlighted
* **HTML5:** Semantic tagging and clean document structure.
* **CSS3:** Custom styling, hover transitions, and variable management.
* **CSS Grid & Flexbox:** Implemented an advanced asymmetric masonry layout for the destination cards, seamlessly aligning elements spanning multiple rows and columns.
* **UI Design:** Focus on whitespace, readable typography, and engaging hero-section imagery.

<br>

## Design Notes
*Note: As this is Phase 1 of the training track, this repository represents a strictly fixed-dimension desktop architecture. Responsive mobile behavior and fluid typography are implemented in Phase 2.*
